# tmux相关操作
+ 在tmux上打开滚动模式：Ctrl + B，然后松开，再按[
+ tmux退出滚动模式：q
+ 查看已有的tmux对话：tmux ls
+ 重新连接名为my_session的会话：tmux attach -t my_session
+ 关闭tmux窗口但不退出tmux会话：Ctrl + B，然后松开，再按D
